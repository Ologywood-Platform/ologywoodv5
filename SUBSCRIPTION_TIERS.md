# Ologywood Subscription Tiers

## Overview

Ologywood offers three subscription tiers designed to serve different user needs: Free for casual users exploring the platform, Basic for active artists and venues managing regular bookings, and Premium for professional operations requiring advanced features and priority support.

## Free Tier

The Free tier provides a solid foundation for users to explore Ologywood's core features without financial commitment. This tier is ideal for independent artists just starting out or venues testing the platform.

### Features Included

Artists on the Free tier can create one limited artist profile with basic information including bio, genres, and contact details. They can create one basic rider template with essential technical and hospitality requirements. The platform allows two booking requests per month, sufficient for casual users to test the booking workflow. Basic contract management includes standard performance contracts with simple signing workflows.

The rider comparison tool on Free tier is limited to comparing two rider templates at a time, helping users understand the feature without full access. Basic rider acknowledgment workflow allows venues to acknowledge riders but with limited modification capabilities. Free users receive basic analytics showing booking metrics and rider acceptance rates.

PDF export is limited to one document per month, and email notifications are sent for critical events only (new bookings, contract signatures). Support access is limited to community forums and email with 48-hour response times. Free users cannot access the API, and team collaboration is limited to the individual account owner.

### Limitations

Free tier users cannot use custom branding, and payment processing is not available. API access is completely restricted. The analytics dashboard shows only basic metrics without trend analysis or detailed reporting. Rider templates are limited to one, and booking requests are capped at two per month to encourage upgrade.

### User Experience

Free tier users see upgrade prompts when they reach feature limits (e.g., "You've used your 2 monthly booking requests. Upgrade to Basic for unlimited requests"). The dashboard displays a banner highlighting premium features with a clear upgrade path. Support is primarily through community forums and email, with a 48-hour response time.

## Basic Tier

The Basic tier ($29/month) targets active artists and venues managing regular bookings. This tier removes most feature limitations while maintaining reasonable usage caps to encourage Premium upgrades for power users.

### Features Included

Basic tier users can create unlimited artist profiles, allowing them to manage multiple acts or personas. They can create up to five rider templates, sufficient for most artists to have variations for different venue types or event sizes. The tier includes 20 booking requests per month, supporting regular booking activity.

Advanced contract management includes custom contract templates, version control, and detailed audit trails. The full rider comparison tool allows unlimited side-by-side comparisons. The complete rider acknowledgment workflow enables venues to propose modifications with full negotiation capabilities. Standard analytics provide insights into booking trends, rider acceptance rates, and team performance.

PDF export allows 10 documents per month, suitable for regular use. Standard email notifications include all booking updates, contract changes, and rider modifications. Support includes email access with 24-hour response times and business hours chat support (9 AM - 6 PM EST, Monday-Friday). Basic tier includes limited API access for basic integrations.

Team collaboration allows up to three team members to access the account with role-based permissions (admin, manager, viewer). Basic custom branding includes logo and color customization for generated documents.

### Payment Processing

Basic tier users pay 2.9% + $0.30 per transaction when processing payments through Stripe, standard for most SaaS platforms.

### User Experience

Basic tier users have a streamlined dashboard with all core features visible. Upgrade prompts are minimal, appearing only when attempting to use Premium-exclusive features. The interface includes team management tools for inviting collaborators. Analytics dashboard shows trend charts and performance comparisons.

## Premium Tier

The Premium tier ($99/month) is designed for professional operations, agencies, and high-volume users requiring maximum flexibility and priority support.

### Features Included

Premium tier provides unlimited everything: artist profiles, rider templates, booking requests, and PDF exports. Advanced contract management includes all Basic features plus custom contract workflows and integration with external legal services. The rider comparison tool has no limitations, supporting comparison of unlimited riders simultaneously.

Premium rider acknowledgment workflow includes priority processing with dedicated support for complex negotiations. Advanced analytics provide detailed insights including predictive analytics, custom report generation, and integration with business intelligence tools. Real-time dashboards show live booking activity and performance metrics.

Email notifications are fully customizable with webhook support for custom integrations. Support includes 24/7 priority phone and email support with 1-hour response time guarantee. Full API access enables custom integrations, white-label solutions, and advanced automation.

Team collaboration is unlimited with granular role-based access control (admin, manager, editor, viewer, custom roles). Full custom branding includes white-label options, custom domain support, and branded client portals for venues.

### Payment Processing

Premium tier users benefit from reduced transaction fees: 2.5% + $0.30 per transaction, saving 0.4% compared to Basic tier.

### User Experience

Premium tier users have a fully customizable dashboard with advanced widgets and custom layouts. The interface includes advanced team management with role templates and permission matrices. Analytics are comprehensive with export capabilities and scheduled report delivery. No upgrade prompts appear, and users have access to a dedicated account manager.

## Feature Breakdown by Category

### Rider Templates & Management

Free tier users can manage one basic rider template with limited customization. Basic tier expands this to five templates with full customization options. Premium tier removes all limitations, allowing unlimited templates with version history and cloning capabilities.

### Booking & Contracts

Free tier supports two monthly bookings with basic contracts. Basic tier allows 20 monthly bookings with advanced contract features including templates and audit trails. Premium tier offers unlimited bookings with full contract customization and legal integration options.

### Analytics & Reporting

Free tier provides basic metrics showing booking counts and rider acceptance rates. Basic tier includes standard analytics with trend charts and performance comparisons. Premium tier offers advanced analytics with predictive insights, custom reports, and business intelligence integration.

### Support & Communication

Free tier has community forum access and email support with 48-hour response times. Basic tier includes business hours chat and 24-hour email support. Premium tier provides 24/7 priority support with phone access and dedicated account management.

### Team & Collaboration

Free tier is single-user only. Basic tier allows three team members with basic roles. Premium tier supports unlimited team members with custom role definitions and granular permissions.

### Integration & Customization

Free tier has no API access or custom branding. Basic tier includes limited API access and basic branding customization. Premium tier offers full API access, white-label options, and custom domain support.

## Upgrade Paths

### From Free to Basic

Free tier users can upgrade to Basic at any time through the Settings â†’ Subscription panel. The upgrade is prorated if initiated mid-month. Upon upgrade, users immediately gain access to all Basic features including the ability to create five rider templates and submit 20 monthly booking requests.

### From Basic to Premium

Basic tier users can upgrade to Premium through the same interface. The upgrade includes all Premium features and priority support. Existing data (profiles, riders, bookings) is automatically migrated with no data loss.

### Downgrade Policies

Downgrading from Premium to Basic is allowed but takes effect at the next billing cycle. Downgrading from Basic to Free is possible but results in feature restrictions. Users are warned about potential data loss when downgrading (e.g., excess riders beyond the tier limit will be archived).

## Billing & Payment

All subscriptions renew monthly on the same date each month. Free tier requires no payment method. Basic and Premium tiers use Stripe for secure payment processing. Users can cancel anytime through the Settings panel, with cancellation taking effect at the next billing cycle.

## Feature Restrictions & Enforcement

The platform enforces tier limitations through UI restrictions and backend validation. Free tier users see disabled buttons for features beyond their tier with helpful tooltips explaining the limitation and upgrade path. When users reach monthly limits (e.g., booking requests), they receive notifications with clear upgrade prompts.

Basic tier users attempting to create a sixth rider template see a modal explaining the limit with an upgrade button. Premium tier users have no feature restrictions and can use all capabilities without limitations.

## Special Offers & Promotions

Annual subscriptions receive a 20% discount (Basic: $278/year, Premium: $950/year). New users receive a 14-day free trial of Premium tier to experience all features. Non-profit organizations and educational institutions receive 50% discount on all tiers.

## Future Tier Enhancements

Planned enhancements include Enterprise tier for organizations with 50+ users, featuring dedicated infrastructure and custom SLAs. Advanced marketplace features are planned for Premium tier, including featured artist listings and priority booking visibility.
